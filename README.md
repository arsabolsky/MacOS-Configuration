# MacOS-Configuration

Configuration for MacOS environment setup.

## 🚀 Quick Start

To set up a new Mac, clone this repository and run the setup script:

```bash
git clone https://github.com/your-username/MacOS-Configuration.git ~/Documents/MacOS-Configuration
cd ~/Documents/MacOS-Configuration
chmod +x setup.sh
./setup.sh
```

## 📦 What's Included?

- **`Brewfile`**: List of CLI tools (git, gh, starship, etc.) and GUI apps (VS Code, iTerm2, Docker, etc.).
- **`.zshrc`, `.gitconfig`, & `ghostty`**: Personal shell, git, and terminal emulator configurations.
- **`macos.sh`**: Script to automate macOS system defaults (Dock, Finder, Keyboard, etc.).
- **`setup.sh`**: Automation script to install Homebrew, Oh My Zsh, dependencies, and apply all configs.


## 🛠️ Customization

- **Brewfile:** Edit this file to add or remove tools and applications.
- **setup.sh:** Add more automation steps (like setting system defaults) here.
